Jawaban Pertanyaan Anda: "Curriculum Learning itu diletakkan setelah Fusion atau di mana?"

Curriculum Learning (CL) BUKAN lapisan di dalam model (*bukan layer Neural Network*). Oleh karena itu, ia tidak berada "setelah Fusion" ataupun "sebelum Fusion". CL berada DI LUAR arsitektur model.

Seperti yang terlihat pada diagram di atas:
  • Model (Kotak Oranye): Tugasnya hanya menerima data → mengekstrak → fusion → klasifikasi → mengeluarkan tebakan (probabilitas).
  • Curriculum Learning (Kotak Hijau): Tugasnya adalah membungkus model tersebut. CL mengamati seberapa meleset tebakan model (Loss). Dari nilai Loss tersebut, CL mengatur ulang antrian pasien, lalu menyuapkan kembali data tersebut ke INPUT model untuk dipelajari ulang pada Epoch berikutnya.
Inilah mengapa CL sering disebut sebagai Training Strategy atau Data Scheduler, bukan arsitektur fisis jaringan.

Fase Pengujian Akhir:
Setelah model selesai dilatih dan dimatangkan oleh Curriculum Learning, tahap pembelajarannya dihentikan (bobot dikunci). Model tersebut kemudian diuji menggunakan dataset tes murni untuk menghasilkan metrik Akurasi, Area Under the Curve (AUC), dan parameter evaluasi klinis lainnya.
